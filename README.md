# Installation

Add the dependency:

```bash
npm i react-native-shake-text
```

## Peer Dependencies

<b><i>Zero Dependency</i></b>

# Usage

## Import

```jsx
import ShakeText from "react-native-shake-text";
```

## Fundamental Usage

```jsx
<ShakeText>Shake Text</ShakeText>
```
